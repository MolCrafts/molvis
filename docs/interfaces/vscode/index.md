# VS Code

The MolVis extension integrates molecular files with VS Code's editor,
workspace, remote-file, command, and settings APIs. It is not a separate
renderer: its webviews host the same MolVis frontend described by the tutorial.

## Choose a surface

| Surface | Use it when |
|---|---|
| Quick View custom editor | One file is the document of record |
| Side-by-side Quick View | You want text and the molecular view together |
| MolVis Workspace | You need the complete UI, multiple loads, analysis, and pipeline tools |
| Activity Bar Home | You are opening recent files or choosing a workflow |
